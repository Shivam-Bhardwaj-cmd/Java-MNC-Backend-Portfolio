# Day 64: Applications of LCS

## Topics Covered
- Shortest Common Supersequence (SCS)
- Minimum Insertions & Deletions to make two strings equal
- Printing the actual LCS string

## Key Concepts
- LCS is a foundation for multiple DP-based string problems.
- Used in file diff tools (e.g., Git), DNA sequence matching, and data comparison systems.
- Time Complexity: O(m * n)
