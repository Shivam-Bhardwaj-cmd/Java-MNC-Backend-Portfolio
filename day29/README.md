# Day 29 — Stack & Queue Combo: Postfix Evaluation, Min Stack & LRU Cache

### Topics Covered:
- Postfix (Reverse Polish Notation) Evaluation using Stack  
- Min Stack: get minimum element in O(1) time  
- LRU Cache using LinkedHashMap  

### Files:
1. EvaluatePostfixDemo.java — Evaluates postfix arithmetic expressions.  
2. MinStackDemo.java — Implements a stack that tracks the minimum element.  
3. LRUCacheDemo.java — Implements a least recently used cache.  

### Key Concepts:
- Stack operations for expression evaluation  
- Auxiliary stack for O(1) min retrieval  
- LinkedHashMap with accessOrder=true for LRU behavior
