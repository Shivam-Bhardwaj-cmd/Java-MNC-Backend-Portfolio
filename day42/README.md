# Day 42 - Graph MSTs and Advanced Applications

## Topics Covered

1. **Prim's Algorithm**
   - Minimum Spanning Tree using priority queue
   - `PrimMSTDemo.java`

2. **Kruskal's Algorithm**
   - MST using Union-Find
   - `KruskalMSTDemo.java`

3. **Dijkstra Algorithm (Optimized)**
   - Single-source shortest path using priority queue
   - `DijkstraWithPriorityQueue.java`

4. **Cycle Detection using Union-Find**
   - Detect cycles in undirected graphs
   - `GraphCycleDetectionUnionFind.java`

## Key Points
- Prim vs Kruskal: trade-offs and when to use which.  
- Union-Find is crucial for Kruskal and cycle detection.  
- Optimized Dijkstra reduces time complexity for sparse graphs.
